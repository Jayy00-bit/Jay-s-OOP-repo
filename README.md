# Using Special Methods in OOP 

This repository contains a Jupyter Notebook demonstrating how to use special (or "dunder") methods in Python's Object-Oriented Programming (OOP). It explains how to use these methods so custom objects can behave naturally and intuitively.

The notebook contains 12 special methods:

- `__init__()` – Constructor
- `__str__()`, `__repr__()` – String representation
- `__len__()` – Length
- `__eq__()` – Equality
- `__add__()`, `__sub__()`, `__mul__()` – Operator overloading
- `__getitem__()` – Item access
- `__setitem__()` – Item assignment
- `__contains__()` – Membership testing
- `__call__()` – Callable objects
- `__iter__()`, `__next__()` – Iteration support
- `__bool__()` – Boolean evaluation
- `__del__()` – Destructor

Every method has a brief explanation with code that has been run to display its functionality

# Accessing the notebook
